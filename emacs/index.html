Emacs

Feb 3rd, 2013

操作

Quit

C-g

コピー

M-w

カット

C-y
C-k　　行末までカット

貼付け

C-y

削除

C-d

取消（Undo）

C-x u

検索

C-s
C-r　　リバース検索

置換

M-%

保存

C-x C-s
C-x C-w　　別名で保存

全選選択

C-x h

その他

バッファリスト

C-x C-b

shellバッファ

M-x shell　　shellバッファ起動
C-x b　　バッファの切り替え

ウィンドウ分割

C-x 2　　上下に分割
C-x 3　　左右に分割

C-x 0　　現在いるウィンドウを閉じる
C-x 1　　全ての分割を解除

評価

C-x C-e　　S式の評価
