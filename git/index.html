Gitコマンド

Feb 3rd, 2013

取消

git reset - -hard HEAD　　前回のcommit直後の状態に戻る。

git commit - -amend -m ”　　前回のコミットをやり直す（git addをしたあとに）

ブランチ

git branch ブランチ名　　ブランチの作成
git checkout ブランチ名　　ブランチの移動
git checkout -b ブランチ名　　ブランチの作成と移動

マージ

git merge マージするブランチ名
